# 🎥 YouTube Video Assistant with LLMs

Aplicação em Python que carrega a transcrição de um vídeo do YouTube e utiliza um modelo de linguagem para responder consultas sobre o conteúdo do vídeo, como resumos, explicações e perguntas específicas.

O projeto suporta diferentes provedores de LLM:

- Hugging Face Hub
- OpenAI
- Ollama

---

## 🚀 Funcionalidades

- Carrega transcrição de vídeos do YouTube
- Suporte a múltiplos idiomas de transcrição:
  - português
  - português do Brasil
  - inglês
- Permite escolher o provedor do modelo em tempo de execução
- Gera respostas com base no conteúdo transcrito
- Pode ser usado para:
  - resumo de vídeos
  - perguntas sobre o conteúdo
  - extração de informações principais
